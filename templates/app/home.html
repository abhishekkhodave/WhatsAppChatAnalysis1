Abstract

WhatsApp become an integral part of day today life to share thought or chat, and organizations also having current focus in the industry is to build a better Chabot enriching the human experience. Comprehension of customer reactions thus becomes a natural expectation. To achieve this, the business chat needs to understand the computational linguistics, context, and tone of the customer. The opinion or sentiment expressed in a document or sentence can be binary (positive, negative. Enabling sentiment analysis on WhatsApp chat not only helps tweak the responses according to user mood but also helps in advance analysis of services and breakdowns. WhatsApp group chat generate large amounts of data every day. Every user sends a lot of textual data over the network. The proposed chat analysis system uses this data to learn and predict the sentiment or the mood of the user at that particular time. Human being can correctly predict the mood of the user by reading his/her chat messages. To enable a system to do a similar task we need to convert this textual data into features which a machine can understand. Once the features are selected the system classifiers are trained using a training set.
